---

### 🧠 Project Overview

AI Career Workspace is a personal productivity tool designed to streamline the job search process by combining traditional software features with AI-powered analysis.

The application centralizes multiple steps of the job application workflow:

- discovering companies
- analyzing job postings
- evaluating personal skill fit
- generating application letters
- tracking submitted applications

The goal was to build a **single workspace for managing the entire application pipeline** while using AI to extract structured insights from job postings.

---

### 🔎 AI Job Link Analysis

![Job Link Analysis](/renereisenhofer/assets/images/CareerWorkspace_AI_analysis.png)

Users can paste a job posting URL and automatically extract structured information from the listing.

The system analyzes:

- role requirements
- technologies
- experience expectations
- skill relevance

It then calculates an **AI-based job fit score** and generates a structured evaluation.

---

### 🏢 AI Company Discovery

![Company Search](/renereisenhofer/assets/images/CareerWorkspace_AI_search.png)

The system can search for companies based on:

- role
- location
- technologies or keywords

Results show potential employers and link directly to their career pages.

---

### 📋 Application Tracking System

![Application Management](/renereisenhofer/assets/images/CareerWorkspace_Applications.png)

A built-in application manager allows users to:

- track submitted applications
- store job links
- manage application statuses
- set follow-up reminders
- store notes about job postings

This turns the tool into a lightweight **personal job CRM**.

---

### ✉️ AI Application Letter Generator

The system can generate tailored application letters based on:

- job description
- extracted skill requirements
- candidate profile
- identified strengths and weaknesses

The goal is to create **context-aware application drafts** that can be quickly refined before submission.
