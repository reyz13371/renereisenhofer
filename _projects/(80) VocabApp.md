# VocabApp

![VocabApp Main Menu](/renereisenhofer/assets/images/VocabAppMainMenu.png)

### 🛠 Tech Stack

- VS Code
- TypeScript
- React Native
- SQLite
- Expo

---

### 📚 Features

- Vocabulary organized into lessons and folders
- Flashcard-based practice mode with progress tracking
- Customizable practice options (shuffle, infinite mode, inverted cards)
- CSV import and export for easy vocabulary management
- Clean, dark-themed mobile UI optimized for daily use

---

### 📌 Project Status

Personal project built to support my own Spanish learning.  
Fully functional mobile app – actively used.
